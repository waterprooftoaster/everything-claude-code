# everything-claude-code

Forked from [affan-m/everything-claude-code](https://github.com/affaan-m/everything-claude-code). The original directory has blown up to become a multi-agent harness. I distilled it back down to be only relevent for claude code.

The `main` branch holds the full collection. Each project-type branch is a curated, portable `.claude/` directory I actually use for that kind of project. 

# Set Up

1. Clone a single branch. Replace "branch-name" with the one you want.

```
git clone -b branch-name --single-branch https://github.com/waterprooftoaster/everything-claude-code
```

2. Then simply move the `.claude` directory into your project root.
